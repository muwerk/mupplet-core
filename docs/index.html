mupplet-core is a collection of hardware applets for the muwerk scheduler

Introduction

mupplet-core implements the following classes based on the cooperative scheduler muwerk:

  • ustd::Light
  • ustd::Switch
  • ustd::DigitalOut
  • ustd::FrequencyCounter
  • ustd::LightsPCA9685
  • ustd::HomeAssistant

Additionally there are implementation for the following helper classes:

  • ustd::LightController
  • ustd::Astro

For an overview, see: mupplet-core readme

Libraries are header-only and should work with any c++11 compiler and and support platforms esp8266 and esp32.

This library requires the libraries ustd, muwerk and requires a platform define.
